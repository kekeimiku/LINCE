# 这是什么？

这是一个指针扫描器，帮助人们找到想要的地址路径以及破解aslr [tutorial](tutorial.md) 。

前往 [release](https://github.com/kekeimiku/ups/releases) 页面下载此程序。

早期开发中，使用方法随时可能有较大变化，可能 `--help` 更新不及时，建议关注 [changelog](changelog.md) 查看更改日志。
